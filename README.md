# 📊 Excel Data Analysis & Dashboard Project

## 📌 Overview

This project demonstrates end-to-end data analysis using Microsoft Excel, covering fundamental to advanced concepts including formulas, data cleaning, dashboards, and automation using VBA.

The project is divided into multiple assignments (Q1–Q8), each focusing on a specific Excel skill required for data analyst roles.

---

## 🧰 Tools & Technologies Used

* Microsoft Excel
* Power Query
* Power Pivot (Data Modeling)
* Pivot Tables & Charts
* Excel Formulas (Arithmetic, Logical, Lookup, Date Functions)
* VBA (Macros & UserForms)

---

## 📂 Project Structure

| File                          | Description                                     |
| ----------------------------- | ----------------------------------------------- |
| Q1. Arithmetic Functions.xlsx | Basic calculations using SUM, AVERAGE, MIN, MAX |
| Q2. Logical Functions.xlsx    | IF, AND conditions for business logic           |
| Q3. Lookup Functions.xlsx     | VLOOKUP, INDEX & MATCH, Named Ranges            |
| Q4. Date Functions.xlsx       | Date extraction & age calculation               |
| Q5. Sort and Filtering.xlsx   | Data sorting & filtering techniques             |
| Q6. Charts.xlsx               | Data visualization using charts                 |
| Q7. Dashboard.xlsx            | Interactive dashboard using Power Query & Pivot |
| Q8. Macro and VBA.xlsm        | Automation using VBA & UserForms                |
